# Database Setup (MySQL) #
This project uses MySQL as its database.

## Install MySQL #
Download and install MySQL Server + MySQL Workbench:
 https://dev.mysql.com/downloads/
During the installation you will be asked to set a root password.
Remember this password, because you will need to enter it in the project.

## Create the Database and Table #
Open MySQL Workbench
Create a new SQL tab
 run the following code that you find in file globo-database.sql

# Working With Node.js + Express + EJS #

Make sure you have **Node.js** installed.
Not yet installed? 

```bash
npm install
```


## To Run ##

### Build the project ###
```bash
npm run build
```


### Start the server ###
```bash
npm start
```

## Go to: ##
```bash
http://localhost:3000
```
